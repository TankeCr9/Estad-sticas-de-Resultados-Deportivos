## Escenario Seleccionado
**Escenario D – Estadísticas de Resultados Deportivos.**
Este proyecto automatiza el procesamiento de datos históricos de partidos de fútbol para auditar la veracidad de las estadísticas, calculando promedios de gol y generando reportes reproducibles de rendimiento.

## Estructura del Repositorio
* `/datos`: Almacena el dataset origen en formato CSV.
* `/scripts`: Contiene el código fuente en Python para el análisis.
* `/resultados`: Guarda los reportes generados y los gráficos estadísticos.

## Instrucciones de Ejecución
Para reproducir el análisis en cualquier entorno local o virtual, ejecute:
`python scripts/analisis_datos.py`
